Створити парсер контенту з баз даних (дампів) + веб-інтерфейс для керування парсером. Можна використати один з наведених фреймворків: Laravel, Symfony 3/4, Phalcon 3.

В інтерфейсі має бути можливість вибору одної, декількох або усіх БД зі списку. Можливість додавання нових дампів(баз) через інтерфейс керування. Список має сформуватись при зчитуванні файлів з папки.
В результаті маємо отримати файл (один або декілька) формату csv/xml/txt, зі статтями з відповідних вибраних баз. Отримати маємо у вигляді посиланнь на скачування в інтерфейсі.
Кожна стаття має складатись з назви та тексту.
Має бути функціональна можливість обʼєднати кілька результуючих файлів в один.
Зі статтей потрібно прибрати будь які посилання та зображення, при цьому потрібно зберегти початкове форматування.
